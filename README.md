### PCB_Quadrotor
- 这是一个PCB四轴飞行器，使用8520空心杯电机和55mm桨，用于科研实验，可作为一个简单的空中移动平台
- 该项目有两个分支，一个上面使用Fly_Control飞控（F103），另一个使用F405作为控制器，搭载F072作为下载器和仿真器。F405的版本在all_in分支之中。



### 特性
- 主控单片机使用STM32F103RCT6
- 板载MPU9250（三轴陀螺仪+三轴加速度计+三轴磁力计）
- 板载MS5611气压计
- 板载飞行器黑匣子W25Q
- 提供HC05蓝牙接口、NRF24L01遥控接口
- 提供I2C接口，便于拓展其他接口
